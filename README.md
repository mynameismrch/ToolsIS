Информационная система: «Web-сайт для поиска видеографов»

Пользователи:
1)	Клиент
2)	Видеограф
3)	Модератор
4)	Администратор

Общие функциональные требования:
1)	Авторизация клиентов/видеографов
2)	Блок создания заказа
  a.	Возможность добавления названия заказа
  b.	Возможность добавления описания заказа
  c.	Возможность добавления фотографий
  d.	Возможность добавления видео
  e.	Выбор даты/диапазона дат
  f.	Добавление цены за услугу
3)	Функция уведомления о принятии заказа видеографом и возможность выбора одного исполнителя
4)	Функция просмотра заказов
5)	Функция просмотра комментариев
6)	Функция просмотра оценок видео
7)	Функция просмотра профилей пользователей
8)	Функция просмотра видео
9)	В подсистеме должна быть реализована функция обработки профилей пользователей
  a.	Функция изменения профилей пользователей
  b.	Функция удаления профилей пользователей
  c.	Функция добавления видео в профиль
  d.	Функция добавления описания в профиль пользователя
10)	Функция калькулятора расчёта цен
11)	Функция получения ссылки на видео-работу
12)	Горизонтальная Live-лента, обновляющаяся путём добавления новых видео-работ
  a.	Возможность удалять собственное видео из Live-ленты
  b.	Возможность прокрутки ленты назад
  c.	Появление информации (Автор, название) о видео, при наведении курсора на него
13)	Возможность комментировать пользователям видео-работы
14)	Возможность оценивать пользователям видео-работы
15)	Окно поиска
  a.	Поиск по тегам
  b.	Поиск по городам
  c.	Поиск по видеографам
  d.	Поиск по заказам
  e.	Поиск по датам
16)	Чат между пользователями
17)	Функция оповещения о поступлении сообщения из чата
18)	Функция составления рейтингов между пользователями
19)	Функция отображения рейтингов
20)	Функция автоматической геолокации пользователя на основе провайдера (ip адрес)
21)	Функция отображения рекламного блока

Функциональные требования у группы «Администратор»:
1)	Возможность добавления пользователя
2)	Возможность редактирования данных пользователя
3)	Возможность удаления пользователя
4)	Возможность премодерации пользователя
5)	Возможность удаления видео
6)	Возможность редактирования названия видео
7)	Возможность удаления видео из Live-ленты
8)	Возможность удаления комментариев
9)	Возможность редактирования комментариев
10)	Возможность удаления заказов
11)	Возможность редактирования заказов
12)	Возможность редактирования информации в профиле пользователя
Функциональные требования у группы «Видеограф»:
1)	Возможность принимать заказы
2)	Функция оповещения о поступлении заказа
3)	Функциональный модуль календаря событий
  a.	Возможность выбора свободной даты для заказов в календаре событий
  b.	Возможность выбора занятых дат в календаре событий
  c.	Возможность удаления выбранных дат
Функциональные требования у группы «Модератор»
1)	Возможность удаления видео из Live-ленты
2)	Возможность удаления комментариев пользователей, входящих в группы:
  a.	Клиент
  b.	Видеограф
  c.	Модератор
3)	Возможность изменения комментариев пользователей, входящих в группы:
  a.	Клиент
  b.	Видеограф
  c.	Модератор
4)	Возможность удаления заказов
5)	Возможность редактирования заказов
6)	Возможность редактирования информации в профиле пользователей
7)	Возможность удаления видео
8)	Возможность редактирования названия видео
9)	Возможность редактирования данных пользователей
10)	Возможность премодерации пользователей
